# Rules Repository

该仓库用于存储自用的代理规则、重写规则以及脚本等。

## 目录结构

- `rule/`：存储代理规则文件的目录。
- `rewrite/`：存储重写规则文件的目录。
- `script/`：存储脚本文件的目录。
